Шаблонылау

Шаблонылар контроллер мен домен логикасын төмендету мен презентация логикасын айырмашылықты түсіндірудің жасырын жолын береді. Шаблонылардың өзі тұтас қолданбалардың HTML-ін қамтып алар, басқа пішіндерге мән беретін жаттықтамаларда пайдалануы мүмкін, міндетті түрде XML. Шаблоныларды көбінеуленген осы “views” деп аталады, бұлып, модель-көрініс-контроллер (MVC) бағдарламалық әрекет жаттықтамасының екінші компонентінің бір бөлігін құрайды.

Ортақ Қолдаулар

Шаблоныларды пайдалануның ең басқа қолдауы - олардың презентация логикасы мен сіздің барлық қолдауларыңыздың арасында жасалатын таза бөлігіді жасау. Шаблонылардың егізгі міндеті - пішіндерленген мазмұнды көрсету. Олар деректерді іздеу, сақтау немесе басқа көбірек қиынды жұмыстар үшін жауапты емесдік. Бұл бетін, табысты код жасауға жолдау береді, алайда өзара іске қосу көлеміне жетпейді, алайда дайыннан көмекші болады, жасаушылар сервер-тізім кодымен (контроллерлер, модельдер) жұмыс істеген, дизайнерлерді де клиент-тізім кодымен (маркап) жұмыс істеген команда ауруында.

Шаблонылар презентация кодының таратуын дайындау жолдасады. Шаблонылар көбінеуленген көмекші көйлекке (views) орналастырылады, әр бірі бірдені танымаланады. Бұл жолдау кодын қайталанатын қатарларга жаттықтамалар деп аталатын көмек көрсетеді. Мысалы, сіздің сайтыңыздың басы мен астарын бір-бірінен алдыңғы және кейінгі көмекшілерді реттейтін шаблонылар түсінілген мәтінді бір реттейге болады.

Соңында, сіз пайдаланатын кітапхана бойынша болмаса, шаблонылар пайдалану арқылы пайдаланушылардан табысты мазмұнды автоматты түрде тымырлап алу жолдасатын болады. Қазір біраз кітапханалар дайыннан айыналмалы перемендер мен функцияларға қатынасу мүмкіндігін беретін “сендбокс” дайындауын ұсынайды.

Тұрау PHP Шаблоны

Тұрау PHP шаблоны - бұл тек нативті PHP кодты пайдаланатын шаблонылар. Олар PHP әсіл тілі болғаны үшін табиғи таңдау. Бұл просто олардың мәнін табу мүмкін, себебі PHP әсіл тілі біздің кодта басқа кодпен, мәселен, HTML кодпен бірге қосуға болатын. Бұл PHP дайындаушылары үшін пайдалы, себебі олардың білуі керек болмайтын жаңа синтаксис жоқ, оларға қол жетімді болатын функцияларды білуді көмек етеді және олардың код редакторларында әмбебапты синтаксис белгілеу және автодайын басқару бар. Сонымен қатар, тұрау PHP шаблонылары кем дайындалатын тұраулама кезеңіне мұқият көмек етеді.

Әзірлік PHP фреймворктері кез келген түрлі шаблон жүйесін қолдайды, оның ішінде көбінеуленген тұрау PHP болады. Фреймворктардың тысында, Plates немесе Aura.View таралулары, мұндаулармен жұмыс істеу үшін көбінеуленген PHP шаблондарымен жұмыс істеу үшін қазір сақталды, басқа жаңа шаблон функционалдарын ұсынады, міндетті түрде кейбірінан басқару, жасау және кеңейту мүмкіндіктерін ұсынады.

Тұрау PHP шаблонының жасыл нұсқасы

Plates кітапханасын пайдалану арқылы.


<?php $this->insert('header', ['title' => 'Пайдаланушы Профилі']) ?>

<h1>Пайдаланушы Профилі</h1>
<p>Сәлем, <?=$this->escape($name)?></p>

<?php $this->insert('footer') ?>

Өріс қолдану арқылы тұрау PHP шаблонының міндетін қолдауы

Plates кітапханасын пайдалану арқылы.


<html>
<head>
    <title><?=$title?></title>
</head>
<body>

<main>
    <?=$this->section('content')?>
</main>

</body>
</html>

<?php // user_profile.php ?>

<?php $this->layout('template', ['title' => 'Пайдаланушы Профилі']) ?>

<h1>Пайдаланушы Профилі</h1>
<p>Сәлем, <?=$this->escape($name)?></p>

Тіркеулі Шаблоны

PHP дайындаушылары ол болса да, жас көлемді жобалау тілі болып ақпаратты, шаблонылауда өте көмекші болмайды. Twig, Brainy, немесе Smarty* төмендетуімен тіркеулі шаблоны, шаблонылаудың жаңа синтаксисін ұсыну арқылы бұл босақты толтырады. Автоматты тымырлап алу, төмендету және жасаулы бағдарламалар жаттығу, шығару жаңалықтарды дайындайтын кезді тіркеулі шаблонымен бірге, жазуы жоспарлы болады және қолдануы үшін ұзақ жағдайды тиісті азайтындар. Тіркеулі шаблоны көбірек тілдердің арасында бөлінуі мүмкін, бұлардың жаңа пәні Mustache көрсеткіші.

*Хотя Smarty автоматты тымырлап алу ұсынады, бұл мүмкіндік Әдепте қосылмаған жағдайда қосылмайды.

Тіркеулі шаблонының жасыл нұсқасы

Twig кітапханасын пайдалану арқылы.

{% include 'header.html' with {'title': 'Пайдаланушы Профилі'} %}

<h1>Пайдаланушы Профилі</h1>
<p>Сәлем, {{ name }}</p>

{% include 'footer.html' %}

Тіркеулі шаблонының өріс қолдауын пайдалану

Twig кітапханасын пайдалану арқылы.

// template.html

<html>
<head>
    <title>{% block title %}{% endblock %}</title>
</head>
<body>

<main>
    {% block content %}{% endblock %}
</main>

</body>
</html>

// user_profile.html

{% extends "template.html" %}

{% block title %}Пайдаланушы Профилі{% endblock %}
{% block content %}
    <h1>Пайдаланушы Профилі</h1>
    <p>Сәлем, {{ name }}</p>
{% endblock %}
